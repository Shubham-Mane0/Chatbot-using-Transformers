# 🤖 Chatbot using Transformers

## 📌 Overview
This project is a simple AI chatbot built using **Hugging Face Transformers (DialoGPT)**.  
It can interact with users and generate human-like responses in real time.

---

## 🚀 Features
- Uses pre-trained transformer model  
- Interactive console-based chatbot  
- Maintains conversation flow  
- Dynamic response generation  
- Exit with `exit` or `quit`  

---

## 🛠️ Tech Stack
- Python  
- Transformers (Hugging Face)  
- PyTorch  

---

## ▶️ How to Run
```bash
pip install transformers torch
jupyter notebook chatbot_transformers.ipynb
